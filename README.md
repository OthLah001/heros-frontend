# Local Demo

## Requirements

You need to install:
* [Node.js](https://nodejs.org/) (> 14.20.0)
* [NPM](https://npmjs.org/)
* [Git](https://git-scm.com/)

## Clone the repo and install the packages

    $ git clone https://github.com/OthLah001/heros-frontend.git
    $ cd heros-frontend
    $ npm i

## Run the server

    $ npm start
